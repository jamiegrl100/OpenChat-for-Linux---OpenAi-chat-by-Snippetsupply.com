# Changelog

All notable changes to **OpenChat for Linux** will be documented in this file.

The format is based on Keep a Changelog, and this project aims to follow Semantic Versioning.

## [1.0.0] - 2026-03-17
### Added
- Initial public release of OpenChat for Linux.
- Desktop client for OpenAI Chat (ChatGPT-style experience) built with Tauri (Rust).
- Efficient chat rendering for long conversations (virtualized message windowing while scrolling).
- Release artifacts:
  - AppImage (amd64)
  - Debian package (.deb, amd64)
  - Tarball (.tar.gz, x86_64)
  - Manifests/specs for Snap, RPM, Nix, and AUR packages.

### Notes
- Some packaging formats may require additional build steps or store submission.
