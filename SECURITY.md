# Security Policy

## Supported Versions

We provide security updates for the latest released version of **OpenChat for Linux**.

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security reports.

Email: **security@snippetsupply.com**

Include:
- A clear description of the issue and impact
- Steps to reproduce (proof-of-concept if possible)
- Affected version(s)
- Your environment (distro, version, desktop, etc.)

We will acknowledge receipt and work with you to validate and resolve the issue.

## Disclosure

We prefer coordinated disclosure. Please allow us reasonable time to investigate and ship a fix before publishing details.
