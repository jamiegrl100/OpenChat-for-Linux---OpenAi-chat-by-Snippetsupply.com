# Support

Thanks for using **OpenChat for Linux** by **SnippetSupply**.

## Where to get help

- Product page: https://snippetsupply.com/
- Support contact: **support@proworkbench.com**

## What to include in your support request

Please include as much of the following as you can:

- App version (e.g., 1.0.0)
- Your Linux distro + version (Ubuntu 24.04, Mint 23.2, etc.)
- Desktop environment (GNOME/KDE/XFCE/etc.)
- Install format you used (AppImage / .deb / tarball / rpm / snap / nix)
- What happened + what you expected
- Any logs you can provide:
  - If launched from a terminal, copy/paste the terminal output
  - If you’re using the AppImage, run it from a terminal and paste the output

## Common quick fixes

- Re-download and verify the checksum (see VERIFYING.md)
- AppImage: make sure it is executable (`chmod +x *.AppImage`)
- .deb: run `sudo apt -f install` if dependencies fail

We usually respond faster when the request includes the details above.
